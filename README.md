# Aarogyam-MERN-App

A full-stack MERN (MongoDB, Express, React, Node.js) application designed to streamline health management processes.

### Technologies Used
* **Frontend:** React, Vite
* **Backend:** Node.js, Express.js
* **Database:** MongoDB

### Features
* User authentication and profile management.
* Real-time data handling for health records.
* Responsive UI for seamless user experience.

### Getting Started
To run this project locally:

1. Clone the repository: 
`git clone https://github.com/Shivansh110641/Aarogyam-MERN-App.git`

2. Install dependencies: 
`npm install`

3. Start the application: 
`node server.js`
